## Fix 2: Async → Sync Playwright API（1 file, 3 tests）

根因：`test_status_bubble.py` 是唯一使用 `async_playwright` + `async def` fixture 的文件。`pytest.ini` 配置了 `asyncio_default_fixture_loop_scope=session`（单元/集成测试需要），导致 pytest-asyncio 无法在 session 级 event loop 内再启动 Runner。其他 3 个 E2E 文件使用 `sync_playwright` + 普通 `def` fixture 和测试方法——无此问题。

修改 `@backend/tests/e2e/test_status_bubble.py`，模式参考 `@backend/tests/e2e/test_student_journey.py`（同步 Playwright API 范例）。

### Task 2.1: 修改 import

```python
# Before
from playwright.async_api import sync_playwright, Page, Browser
# （注意：如果之前导入的是 async_playwright，改为 sync_playwright）

# After
from playwright.sync_api import sync_playwright, Page, Browser
```

### Task 2.2: 修改 browser fixture

```python
# Before
@pytest.fixture(scope="module")
async def browser():
    async with async_playwright() as p:
        browser = await p.chromium.launch(headless=True)
        yield browser
        await browser.close()

# After
@pytest.fixture(scope="module")
def browser():
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        yield browser
        browser.close()
```

### Task 2.3: 修改 page fixture

```python
# Before
@pytest.fixture
async def page(browser: Browser):
    context = await browser.new_context()
    page = await context.new_page()
    yield page
    await context.close()

# After
@pytest.fixture
def page(browser: Browser):
    context = browser.new_context()
    page = context.new_page()
    yield page
    context.close()
```

### Task 2.4: 修改测试方法——去掉 async/await

将类中 3 个测试方法全部从 `async def` 改为 `def`。将方法体内的 `await page.goto(...)` → `page.goto(...)`，`await page.wait_for_timeout(...)` → `page.wait_for_timeout(...)`，其他 `await` 同理去除。

```python
# Before
async def test_chat_page_loads(self, page: Page):
    await page.goto(f"{BASE_URL}/", ...)
    await page.wait_for_timeout(...)

# After
def test_chat_page_loads(self, page: Page):
    page.goto(f"{BASE_URL}/", ...)
    page.wait_for_timeout(...)
```

### Task 2.5: 验证

```bash
docker compose exec backend python -m pytest tests/e2e/test_status_bubble.py -v --tb=short 2>&1 | tail -15
```
