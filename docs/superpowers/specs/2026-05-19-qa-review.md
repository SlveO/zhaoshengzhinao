### 3.4 边界与错误处理

```
□ DeepSeek API 不可用 → WS 返回 error 消息，不崩连接
□ WebSocket 断连 → 前端自动重连（指数退避）
□ 无效 JWT → 401 + 前端跳转登录
□ 模块未开通 API → 403
□ 无效 Excel 上传 → 校验错误返回
□ 空消息发送 → 被 ContentLengthGuard 跳过
```

### Playwright 浏览器测试

```python
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    b = p.chromium.launch(headless=True)
    page = b.new_page()

    # Admin login flow
    page.goto('http://localhost:3001?tenant=scnu', wait_until='networkidle')
    page.locator('input').nth(0).fill('admin')
    page.locator('input[type="password"]').fill('admin123')
    page.locator('button[type="submit"]').click()
    page.wait_for_timeout(3000)
    assert '/login' not in page.url, "Login failed"

    # Navigate to different pages
    for path in ['/funnel', '/profile', '/brand', '/knowledge', '/insights', '/agent-settings']:
        page.goto(f'http://localhost:3001{path}', wait_until='networkidle')
        page.wait_for_timeout(1000)
        print(f'{path}: OK')

    b.close()
```
