核心结论

  1. B2B模式在商业维度全面优于B2C — LTV提升4000+倍、获客窗口从45天扩展到全年、护城河从中等升至极深。B2C模式的三个结构性缺陷被一次性解决。
  2. 招生SaaS是真实且增长的蓝海市场 — 生源下降 + 新高考改革 + 教育数字化三轮驱动。中国教育在线「招生E答」是最接近的竞品，但产品深度存在代际差距。
  3. 护城河深度远超B2C — 系统集成 + 历史数据积累 + 院校间数据网络效应 = 极高替换成本。一旦院校将招生系统嵌入微信生态、积累了一届学生的完整画像数据，供应商锁定效应极强。
  4. 盈亏平衡门槛极低 — 仅需28所付费院校（占1,270所本科院校的2.2%）即可盈亏平衡。Year 2可实现盈利。
  5. 核心风险从"市场存在性"转为"执行能力" — B2C的风险是"这个市场能成立吗？"B2B的风险是"我们能执行好吗？"后者可控得多。
  6. B2B + B2C 双引擎是最优解 — 以B2B为收入核心、B2C为数据飞轮，两者互为放大器。
  7. 时间窗口友好 — 不同于B2C的45天窗口，B2B没有季节性压力。2026年慢工出细活，目标2027年招生季验证。

1. 多租户架构改造 → 2. 签约3所试点院校 → 3. 白标小程序 + 管理端 → 4. 2027招生季实战 → 5. 规模化增长
