Stage 1

问题与市场验证

3D 场景价值模型：刚性 × 频率 × 可自动化

📌 刚性 · 9/10

选错学校/专业 = 4年时间成本 + 职业偏离。家长已用线下咨询 ¥3000-50000/次证明这是刚需。权重 0.4

⏱ 频率 · 3/10

每年一次，窗口期仅 45 天。这是最大商业风险——无法做订阅制，LTV 天然有天花板。权重 0.3

🤖 可自动化 · 8/10

分数匹配 100% 可自动化；兴趣挖掘 70%+ 可自动化；主观判断部分需人参与但 AI 可结构化引导。权重 0.3

加权得分: 0.4×9 + 0.3×3 + 0.3×8 = 6.9/10

问题真实且痛苦，低频次是硬伤，需用产品矩阵延伸来对冲

风险矩阵

🚨 LLM 幻觉导致错误志愿建议 致命 · 中概率
可能让公司一夜归零。缓解: 每条推荐强制附带数据来源URL+年份；"辅助决策"定位，不做确定性断言。
⚠ 微信小程序审核不通过/下架 高严重 · 中概率
提前准备 ICP 备案 + 教育信息服务机构资质；保留 H5 降级渠道。
⚠ 数据隐私合规 (PIPL) 高严重 · 中概率
加密存储用户敏感数据；提前合规审查；隐私政策透明；提供数据导出和删除功能。
ℹ 获客窗口极短 (45天) 中严重 · 高概率
提前 3 个月预热；预注册锁定用户；内容营销提前布局。
ℹ LTV 天花板 中严重 · 确定
开发高一选科、考研/留学延伸产品；B端年费制提供 ARR。

🎯 最高优先级三件事

1. 微信小程序

没有它 = 没有中国 B2C。2026年7月前完成 v1（对话+画像+推荐，不含支付）。当前 React Web 应用在中国市场等于不存在。技术方案: Taro/uni-app 迁移或原生框架，复用后端 API。

2. 收集真实对话数据

用 2026 高考季找 20-50 个高三毕业生，完整使用产品。记录每一轮对话——哪些话术有效？画像准确率自评？这是话术库训练集，竞争对手无法复制的资产。

3. 数据 3 省→10 省

当前 162 校/3 省仅覆盖 4.6% 考生。10 省覆盖 60%+。Scrapy+Playwright 基础设施已有，核心: 各省适配 + 标准化 + Chroma 重新索引。2-3 个月全职。

核心结论

  1. 问题真实，市场够大 — 1300 万考生 × 每年重复，刚性付费已由线下咨询市场验证
  2. 单位经济优秀 — 85% 毛利率，5400 付费用户即盈亏平衡
  3. 技术有壁垒但不够深 — 心理学对话+证据累积是差异化，数据必须扩展到 10+ 省
  4. LTV 天花板是最大软肋 — 必须通过产品矩阵延伸和 B 端年费对冲
  5. 时间窗口明确 — 2027 年 6 月是第一个商业验证点
  6. 头号风险不是竞争而是自身 — LLM 幻觉给出错误建议可能导致公司一夜归零
