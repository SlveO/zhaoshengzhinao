# 高考志愿智能助手 — 商业可行性分析报告

> 评估日期: 2026-05-15 | 评估对象: Gaokao Agents MVP | 分析方法: 5 阶段商业分析框架

---

## 执行摘要

**商业可行性综合评分: 7.2/10 —— 可以商业化，但有两个必须解决的核心问题。**

本产品以"AI 心理学对话 × 真实录取数据 RAG"的差异化定位，填补了高考志愿填报市场中"高技术壁垒 × 适中价格"的空白象限。问题真实且痛苦（刚性 9/10），单位经济健康（毛利率 85%，5400 付费用户即可盈亏平衡），技术壁垒存在但不是不可逾越。

**两个核心风险**: (1) 没有微信小程序，在中国 B2C 市场等同于不存在；(2) 高考是单次事件，LTV 天然有天花板，必须通过产品矩阵延伸（选科→志愿→考研→留学）延长用户生命周期。

**时间窗口**: 2026 年高考季已来不及。目标应是 2027 年 6 月。这 13 个月的缓冲期正好用于: 完成微信小程序、扩展数据覆盖至 10 省、用今年高考季收集真实对话数据打磨话术库。

## Stage 1: 问题与市场验证

### 1.1 3D 场景价值模型评分

| 维度 | 评分 | 权重 | 依据 |
|------|------|------|------|
| **刚性** | 9/10 | 0.4 | 选错学校/专业 = 4年时间成本 + 职业路径偏离。家长已用行动证明付费意愿——线下咨询 3000-50000 元/次 |
| **频率** | 3/10 | 0.3 | 每年一次，窗口期仅 45 天（6月下旬-7月底）。这是最大商业风险——无法做订阅制 |
| **可自动化** | 8/10 | 0.3 | 分数位次匹配 100% 可自动化；兴趣挖掘 70%+ 可自动化；主观部分（家庭协商）需人参与但 AI 可结构化引导 |

**加权得分**: 0.4×9 + 0.3×3 + 0.3×8 = **6.9/10**

### 1.2 痛点验证

每年 1300 万高考考生面临三个核心困境:

1. **信息过载与盲目跟风**: 数千所院校、数百个专业，"热门"标签误导选择
2. **家长意愿与个人兴趣冲突**: 父母的务实期望与学生内心热爱矛盾
3. **只看分数不看人**: 市面上大多数工具只是"分数-院校"的机械匹配，完全忽视兴趣倾向、价值观和职业人格

### 1.3 现有替代方案缺口

| 方案 | 价格 | 核心缺陷 |
|------|------|---------|
| 免费分数匹配工具（高考直通车、掌上高考） | 0 | 纯机械匹配，无兴趣分析，无数据溯源 |
| 人工咨询师 | ¥3,000-50,000 | 质量参差、不可规模化、依赖个人经验 |
| KOL 博主（张雪峰） | 免费（变现靠流量） | 一对多广播，无法个性化 |
| 百度/知乎/贴吧 | 0 | 信息碎片化、可靠性无法验证 |

**本产品填补的空白**: 高技术壁垒（AI 心理学对话 + RAG）+ 适中价格（¥198-398），这个象限目前无人占据。

### 1.4 验证结论

**问题真实且痛苦，置信度: 高。** 不存在无市场竞争，但存在品类创新机会。低频次是硬伤，需要在商业模式中通过产品矩阵延伸来对冲。

## Stage 2: 目标客户与竞争格局

### 2.1 客户分层（按付费意愿）

| 层级 | 客户画像 | 年规模 | 付费意愿 | 核心痛点 |
|------|---------|--------|---------|---------|
| **Tier 1** | 一线城市中产家长（北上广深），年教育支出 1-5 万 | ~120万家庭 | 高 (¥500-3000) | 信息过载、选择焦虑、需"数据背书" |
| **Tier 2** | 新一线/省会城市，有付费能力但缺优质咨询资源 | ~350万 | 中 (¥200-1000) | 信息不对称，知道要选但不知道怎么选 |
| **Tier 3** | 三四线/县城，极度缺乏专业指导 | ~700万 | 低但量大 (¥50-200) | 几乎是空白市场，免费获客→试用转化 |
| **B端-学校** | 高中生涯规划课老师，人均管 500+ 学生 | ~15万所高中 | 中高 (¥2-10万/校/年) | 政策要求开展生涯教育，缺乏工具 |
| **B端-教育局** | 市/县级教育局 | ~300个地级市 | 高 (¥10-50万/年) | 政策合规、数据安全、覆盖辖区 |

### 2.2 TAM / SAM / SOM

```
TAM:  1300万考生 × ¥300均价 = ¥390亿/年 + B端 ¥50亿 ≈ ¥440亿
SAM:  10省覆盖(约700万考生) × 仅B2C = ¥21亿/年
SOM:  3年目标 — 30万注册 × 8%付费 × ¥350 ARPU + B2B
     ≈ ¥1090万/年
```

**自下而上交叉验证**: 有智能手机 + 愿意为教育付费 + 对AI工具有信任感的家庭 ≈ 300 万。5% 转化 × ¥300 = **¥4.5 亿真实可触及市场**。

### 2.3 竞争定位矩阵

```
                    高技术壁垒
                          │
          ★ Gaokao Advisor│  ● 猿辅导/作业帮
          (心理学+RAG     │    (有AI但副业)
           证据溯源)      │

                          │    影响力大不可规模化
    低价 ─────────────────┼────────────────── 高价
      ● 高考直通车        │  ● 线下咨询 ¥3K-5万
      ● 掌上高考          │    质量不稳定
      (免费,纯分数匹配)    │
                          │  ● 百度贴吧/知乎
                          │    信息碎片化
                    低技术壁垒
```

**右上象限（高技术×适中价格）是空白市场。**

### 2.4 护城河评估

| 护城河类型 | 当前强度 | 评估 |
|-----------|---------|------|
| 数据网络效应 | 🟡 中 | 182K 文档/162 校/3 省。需扩展到 10+ 省才有壁垒 |
| 技术壁垒 | 🟢 中高 | LangGraph 4阶段对话 + 证据累积 + 盲区检测——不是加 prompt 能复制 |
| 切换成本 | 🟡 中 | 深度画像建立后有粘性，但高考是一次性的 |
| 品牌信任 | 🔴 弱 | 当前为零，教育领域信任是最高壁垒 |
| 独占资源 | 🟡 中 | 教育局官方合作可形成独占 |

## Stage 3: 收入模式与单位经济

### 3.1 B2C Freemium 定价

| 层级 | 价格 | 内容 | 预期转化率 |
|------|------|------|-----------|
| 免费版 | ¥0 | L1 基础画像 + 3 所推荐 | — |
| 标准版 | ¥198 | 完整 4 阶段对话 + L2 画像 + Top-20 推荐 + 录取概率 | 5% |
| 高级版 | ¥398 | L3 深度画像 + 无限推荐 + 职业路径分析 + PDF 报告 | 2% |
| 家庭版 | ¥598 | 2-3 个学生档案 + 家庭共享仪表盘 | 0.5% |

**定价锚点**: ¥198 < 一本教辅书；¥398 < 线下咨询的 1/10；¥598 < 一次课外辅导课。

### 3.2 B2B 第二引擎

| 产品 | 定价 | 目标 |
|------|------|------|
| 学校单校许可 | ¥29,800/年 | 高中生涯规划课 |
| 区域教育局包 | ¥198,000-498,000/年 | 市/区级教育局 |
| 白标 API | ¥0.5-2/次查询 | 第三方教育平台 |

### 3.3 单位经济（Year 2 稳态）

| 指标 | 数值 | 说明 |
|------|------|------|
| CAC | ¥12-18 | 抖音内容 + 微信裂变，以自然流量为主 |
| ARPU | ¥310 | 各层级加权平均 |
| 毛利率 | 85% | 主成本 = DeepSeek API + 云服务器 |
| LTV | ¥310 | **核心风险: LTV = 单次购买，无复购** |
| LTV:CAC | 17:1 - 26:1 | 看似好——实际是因为 CAC 低而非 LTV 高 |
| 回本周期 | 即时 | 一次性购买模式 |

**LTV 天花板破局路径**:
1. 向前延伸: 高一选科 + 高二规划 → 提前 2 年触达
2. 向后延伸: 考研 + 留学 + 职涯初阶 → 复用画像资产
3. B 端收入: 学校年费制 → 提供真正的 ARR

### 4.3 盈亏平衡点

```
月度固定成本: ¥117,000
单用户贡献毛利: ¥310 × 85% = ¥263

盈亏平衡点 = 117,000 × 12 / 263 = 5,338 付费用户/年
```

**结论: 仅需 5,400 付费用户（1300万考生的 0.04%）即盈亏平衡。**

## Stage 5: GTM 策略与下一步

### 5.1 时间线

**2026 年高考季已来不及。目标: 2027 年 6 月。**

```
2026 Q2 (5-7月) — 基础建设
├── 6省数据覆盖（江苏、浙江、四川、湖北、山东、湖南）
├── 微信小程序开发启动
├── 找 20-50 个高三毕业生 Beta 测试
└── 收集真实对话 → 优化话术库

2026 Q3 (7-9月) — 产品验证
├── 微信小程序 v1 上线（不收费）
├── 抖音/小红书账号启动
├── 接触 3-5 所广东高中合作
└── "考后反思"窗口内容营销

2026 Q4 (10-12月) — 增长预备
├── 10省数据覆盖
├── 接入微信支付
├── 小程序预注册开启
├── B端签约 5-10 所学校
└── 目标: 预注册 2 万

2027 Q1 (1-3月) — 获客加速
├── 高三家长焦虑高峰期 → 内容营销全力推进
├── Freemium 漏斗激活
└── 目标: 累计注册 8 万

2027 Q2 (4-5月) — 决战前夜
├── 5月1日前系统全面压力测试
├── 预注册 push: "你的画像已就绪"
└── 目标: 累计注册 15 万

2027 6-7月 — 收获季
└── 目标: 付费 6,000+，收入 ¥180万+
```

### 5.2 获客漏斗

```
抖音/小红书曝光  500万次
      ↓ 5%
微信小程序落地页  25万访客
      ↓ 30%
免费注册用户      8万
      ↓ 15%
完成深度对话(L2)  1.2万
      ↓ 50%
付费用户          6,000 → ¥186万 B2C
```

**漏斗关键**: 产品自身的对话体验就是最好的销售——当学生看到 AI 真正理解自己时，家长付费意愿大幅提升。

### 5.3 最高优先级三件事

**1. 微信小程序 — 没有这步，一切归零**

当前 React Web 应用在中国市场等于不存在。必须在 2026 年 7 月前完成微信小程序 v1（Taro/uni-app 迁移或原生框架）。最低可用: 对话 + 画像 + 推荐，不含支付。两个月，一人可完成。

**2. 用 2026 高考季收集真实对话数据**

找 20-50 个真实广东高三毕业生，在知道分数后完整使用产品。记录每一轮对话——哪些话术有效？画像准确率自评？这是话术库的训练集和护城河的起点，竞争对手无法复制。

**3. 数据从 3 省扩展到 10 省**

162 所学校/3 省覆盖仅 60 万考生（4.6%）。扩展到 10 省覆盖 800 万考生（60%+）。Scrapy + Playwright 基础设施已有，核心工作: 各省适配 → 数据清洗标准化 → Chroma 重新索引。预计 2-3 个月全职。

### 核心结论

1. **问题真实，市场够大** — 1300 万考生 × 每年重复，刚性付费意愿已由线下咨询市场验证
2. **单位经济优秀** — 85% 毛利率，5400 付费用户即盈亏平衡，LTV:CAC 健康
3. **技术有壁垒但不够深** — 心理学对话+证据累积是差异化，但数据覆盖必须扩展到 10+ 省才能构筑真正壁垒
4. **LTV 天花板是最大软肋** — 必须通过"选科→志愿→考研→留学"产品矩阵延伸，以及 B 端年费收入对冲
5. **时间窗口明确** — 2027 年 6 月高考季是第一个商业验证点，2026 年是"建设+打磨"年
6. **头号风险不是竞争而是自身** — LLM 幻觉给出错误建议可能导致公司一夜归零，必须从第一天起建立"数据溯源+辅助决策"的产品原则
